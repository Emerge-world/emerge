# 13 — Future Work or Ideas

## Agent Memory Management

Generate a memory system that can store and retrieve information without cluttering the prompt of the agent. This could involve creating a separate database or using an external tool to manage the agent's memory.

## Bug with Innovation Pickup

Agents taking `pickup` action as an innovation action, which is not intended. This could be due to a misinterpretation of the action space or a bug in the code. It would be important to investigate this issue and ensure that agents are correctly interpreting the action space.
